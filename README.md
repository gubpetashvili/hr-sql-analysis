# HR Workforce & Salary Intelligence Analysis

This project explores and analyzes an HR employee database using advanced SQL techniques.  
The goal is to extract meaningful insights about workforce structure, salary distribution, managerial responsibilities and department-level performance.

The analysis demonstrates practical data analysis skills using SQL in a real-world HR scenario.

---

## Project Overview

Organizations rely on workforce analytics to understand employee distribution, compensation structure and managerial efficiency.

This project analyzes HR data to answer key business questions such as:

- Which departments have the highest average salaries?
- Which managers oversee the largest teams?
- How are salaries distributed across departments?
- Which employees earn above department averages?
- How does employee tenure relate to compensation?

## Key Analysis Areas

### Workforce Structure Analysis
Analyzing employee distribution across departments and managerial hierarchies.

### Salary Intelligence
Exploring salary distribution, department averages and identifying high-earning employees.

### Managerial Effectiveness
Evaluating managers based on the number of employees they supervise.

### Department Performance
Comparing departments based on salary budgets and employee counts.
